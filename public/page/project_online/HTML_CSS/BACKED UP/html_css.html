HTML et le CSS

Le HTML (HyperText Markup Language)

Le HTML est un langage de balise utiliser pour les page Web. Voici se que j'ai compris :

Pour commencer chaque balise a une catégorie (ex : < div > = block ; < p > = paragraph ; table = un tableau ; ...)
Ensuit on peux rajouter des éléments a set balis pour la définir ou lui donner des odre : (ex : < div id="Sa_défini"> ; < img src="une image" height="sa hauteur"> ; ...)
Il faut toujour fermé les balise dans l'ordres : < A > < B > < /B > < /A > et non < A > < B >< /A > < /B >

Le CSS (Cascading Style Sheets)

Le Css permet de rajouter le style de la page (sa charte graphique), de la couleur des forme même des annimation.

Quelque astuse pour le css :

  1. Les class et les id (on parlera de la class mais sa vaut pour l'id):
    • On peut apliquer du code que a un élémen ou plusieur d'une classe ou d'une id nous avons plusieur fasson de faire :
      1. La plus simple sais de mêtre la class puis de maitre le nom d'un élément comme par exemple :
        .class p{~~~~}
        Ceci dit a tout les balise < p > qui on comme parent ou grand parent etc d'apliquer le code.
      2. La segond permet de definir particulièrement se con vise, on vas lui demender de le des spésification bien présis :
        .class div p {~~~~}
        Ici on demende de modifier le style de tout les balise < p > qui sont dans une balis < div > si on rajout un < h1 > dèrière le < p >on lui demendera cet fois si de modifier le style de tout les < h3 > qui se trouve dans une balise < div > et < p > .
      3. La troisième et dergnière sais la virgule avec une vergule entre des élément on leur demende davoir le même style : .class div, p {~~~~}
        Ici nous changon tout les p et les div qui apartiène a la class sais a dire que si il on la clase n'import ou dans leur ièrachi il chage de style.
    • Les appel :
      • On appelle une class avec un . puis le nom de la class :
        .class {~~~~}
      • On appelle une class avec un # puis le nom de la l'id :
        #id {~~~~}
      • Nous avons aussi un appel généraliser qui définis des terme pour tout les élément sais * :
        * {~~~~}
    • On les definie :
      • On définie une class avec le mont "class" puis le nom de la class :
        class="nom_de_la_class"
      • On définie une class avec le mot id puis le nom de la l'id :
        id="nom_de_id"
  2. Mètre des élément aligner :
    • Il suffie de maitre display a "flex" dans la div ou vous vouler que les élément soit alligner.
