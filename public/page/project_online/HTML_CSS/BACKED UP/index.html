Accueil

Rapelle du sujet :

Dans un éditeur de texte, comme Notepad++ ou jEdit, ouvrir index.html et style.css
Précaution : Attention à déclarer dans votre document HTML que l’encodage est UTF-8 ainsi que lorsque vous sauvegardez votre ﬁchier.
Utiliser le template Transparent Blue que vous personnalisez ou un autre (voir free-css.com attention les plus récents sont très complexes) Un même css s’applique à toutes les pages, enregistré sous style.css.
  1. Créer une page d’accueil index.html qui vous présente;
  2. Créer une seconde page sur une personnalité du numérique. On la nommera personnalite.html.
  3. Créer une troisième page lexique.html, comportant un lexique informatique. Déﬁnir les acronymes http, html, css, url, TCP, IP, UDP, OSI mais aussi les notions de protocole, serveur, routeur, internet, ethernet, wiﬁ, datagrame, les couches du modèle OSI, l’encapsulation et la décapsulation ...
  4. Créer une quatrième page html_css.html, sur ce que vous avez appris des balises html et les astuces concernant le css. On introduira un tableau dans un élément table (une recherche à faire).
Chaque page contiendra au minimum des titres de diﬀérents niveaux, une image, un paragraphe de texte, une liste ordonnée ou non, un texte mis en évidence (gras, italique) .... Toutes les pages sont reliées entre elles et la navigation ne fait sans impasse (aucun besoin de ←).
Enregistrer tous vos ﬁchiers dans un dossier nommé PN_siteweb ou PN sont vos initiales. Compresser ce dossier pour le déposer dans mon casier.
